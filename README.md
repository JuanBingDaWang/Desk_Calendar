11.25更新说明：
1. 优化了代码，使接口的调用更加统一
2. 修复了记事本窗口大小重启时会复原的问题
3. 优化了“每日聚焦”的显示
4. 现在双击日历中的事件是编辑而不是新建了
5. 备忘录现在可以自动保存，还可以Ctrl+S保存

7. 日历中的事件现在报错在通用的日历文件（Events.ics）中，方便导出到其他日历软件中使用
   
## 核心功能：
1. 可以固定在电脑桌面上，同时不占用任务栏，也做了一些防误触设计。
2. 自带一个备忘录，可以代替电脑的便笺功能。
3. 事件可以设置重复、提醒、提醒可以延迟提醒（和微信学的）。
4.事件可以按照优先级和开始时间排序。
5.可以自定义背景颜色、文字颜色、字号等。
6.使用onedrive、百度网盘等自动同步云盘，可以在多个设备上同步日程。
