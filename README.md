## 核心功能：
1. 可以固定在电脑桌面上，同时不占用任务栏，也做了一些防误触设计。
2. 自带一个备忘录，可以代替电脑的便笺功能。
3. 事件可以设置重复、提醒、提醒可以延迟提醒（和微信学的）。
4.事件可以按照优先级和开始时间排序。
5.可以自定义背景颜色、文字颜色、字号等。
6.使用onedrive、百度网盘等自动同步云盘，可以在多个设备上同步日程。
